30// Define this to disable ASM, intrinsics and built-ins. The library code will
31// not explicitly include SSE2 (and above), NEON, Aarch32, Aarch64, or Altivec
32// (and above). Note the compiler may use higher ISAs depending on compiler
33// options, but the library will not explicitly use the ISAs. When disabling ASM,
34// it is best to do it from config_asm.h to ensure the library and all programs
35// share the setting.
