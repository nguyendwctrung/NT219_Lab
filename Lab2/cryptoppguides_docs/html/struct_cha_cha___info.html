StaticAlgorithmName returns the algorithm's name as a static member function.

Bernstein named the cipher variants ChaCha8, ChaCha12 and ChaCha20. More generally, Bernstein called the family ChaCha{r}. AlgorithmName() provides the exact name once rounds are set.
